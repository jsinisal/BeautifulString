BeautifulString

BeautifulString is a high-performance Python C extension module designed for advanced string manipulation and parsing using format-string-based syntax. Inspired by the functionality of C's sscanf, this module provides powerful utilities for extracting structured data from strings with high speed and flexibility.

Features

  • strfetch(input_str, input_slices)
    Extracts substrings using Python-style slice syntax (e.g. [::-1], [2:], [:5]).
  • strscan(input_str, format_str, return_type="list", dict_keys=None)
    Parses tokenized input based on a format string and returns extracted values.
    • Supports specifiers: %s, %d, %u, %f, %lf, %x
    • Supports named fields using syntax like %s(name)
    • Supports quoted strings with escape sequences (e.g. "Hello\nWorld")
    • Return types: list, tuple, dict, or tuple_list
  • strmatch(input_str, format_str, return_container="list")
    Matches a string exactly using a format-based template.
    • Same specifier support as strscan, with strict full match requirement
    • Supports named fields and quoted strings
  • strsearch(input_str, format_str, start_index=0, max_matches=-1, return_container="list")
    Searches a string for repeated occurrences of a format-based pattern.
    • Partial match mode with multi-hit capability
    • Returns a list of values (dicts, tuples, etc.) depending on return_container

Installation

python setup.py build_ext --inplace

Example Usage

import BeautifulString as bs

s = "User: Alice Score: 99 User: Bob Score: 88"
pattern = "User: %s(name) Score: %d(score)"

matches = bs.strsearch(s, pattern, max_matches=2, return_container="dict")
print(matches)  # [{'name': 'Alice', 'score': 99}, {'name': 'Bob', 'score': 88}]

info = bs.strmatch("X: foo Y: 42", "X: %s(x) Y: %d(y)", return_container="tuple")
print(info)  # ('foo', 42)

scan_result = bs.strscan('Name: "Alice Smith" Age: 30', "Name: %s(name) Age: %d(age)", return_type="dict")
print(scan_result)  # {'name': 'Alice Smith', 'age': 30}

slice_result = bs.strfetch("abcdefgh", "[::-1]")
print(slice_result)  # 'hgfedcba'
