BeautifulString Module

This module provides advanced string parsing utilities inspired by C-style format strings. It includes functions for string token scanning, pattern matching, and flexible formatting extraction.

Function: strfetch(input_str, input_slices)

Extract substrings from input_str using Python-like slice notation given in input_slices.

Example:

s = "Hello, World!"
result = bs.strfetch(s, "[:5], [7:], [::-1]")
# Returns: ['Hello', 'World!', '!dlroW ,olleH']

Function: strscan(input_str, format_str, return_type='list', dict_keys=None)

Tokenizes the input string and parses fields according to the format string.

return_type can be one of: 'list', 'tuple', 'dict', or 'tuple_list'.

Supported format specifiers:

  • %s – String (quoted or unquoted)
  • %d – Integer
  • %u – Unsigned integer
  • %f – Float
  • %lf – Double precision float
  • %x – Hexadecimal

Example:

s = 'Name: "Alice" Age: 30'
bs.strscan(s, "Name: %s(name) Age: %d(age)", return_type="dict")
# Returns: {'name': 'Alice', 'age': 30}

Function: strmatch(input_str, pattern_str, return_container='list')

Performs strict format-string-based matching of the entire input string.

Example:

s = "X: foo Y: 42"
bs.strmatch(s, "X: %s(x) Y: %d(y)", return_container="dict")
# Returns: {'x': 'foo', 'y': 42}

Function: strsearch(input_str, pattern_str, return_container='list', start_index=0, max_matches=-1)

Searches input_str for pattern occurrences based on pattern_str, returning all matches.

Example:

s = "User: Alice Score: 99 User: Bob Score: 88"
bs.strsearch(s, "User: %s(name) Score: %d(score)", return_container="dict", max_matches=2)
# Returns: [{'name': 'Alice', 'score': 99}, {'name': 'Bob', 'score': 88}]
